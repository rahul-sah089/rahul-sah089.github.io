Super in Java

A subclass can make a call to the superclass method,variables and sonstructor using super keyword.

Using super keyword a user can call superclass constructor method.

Using super() statement we can call either zero argument constuctor or parameterized constructor.

If super() statement is been used to call the constructor of the superclass than it should be the 1st statement of th constructor body and there should not be more than one super() method in the body.

1).super.variableName refers to the variable of the variable of the parent class

Below shows the example of how the super keyword can be used to invoke the variable name of the parent class.

//Parent class or Superclass
 class Parentclass{
int num=100;
 }
//Child class or subclass
 class Subclass extends Parentclass{
int num=110;
 void printNumber(){
 //Super.variable_name
 System.out.println(super.num);
 }
 public static void main(String args[]){
Subclass obj= new Subclass();
 obj.printNumber();
}
 }

Output

100

As you can see by using super.num we can access the num variable of the parent class.

2). super() to invoke constructor of parent class

using super() method we can access the constructor method of the it's parent's class. First we will see the default behaviour of the super() method.When we create the object of the subclass the constructor method of the superclass get invoked and then it invoked the constructor of it own. It happens because the compiler add the default constructor(super()) and the first line of the subclass constructor method.

Let see an example of the default invocation of the parent's class constructor

Sample Code

class Parentclass{
Parentclass(){
 System.out.println("Constructor of the Parent Class");
 }
 }
class SubClass extends ParentClass{
SubClass(){
 /* Compiler adds super() here at the first line
* of this constructor implicitly
 */
 System.out.println("Constructor of the SubClass");
 }
 SubClass(int num){
 /* Compiler adds super() here at the first line
* of this constructor implicitly
 */
 System.out.println("Constructor of the SubClass");
 }
 void display(){
 System.out.println("Hello");
 }
 public static void main(String[] args){
 System.out.println("Main Method Started");
 // Creating object using default constructor
 SubClass obj1 = new SubClass();
 //Calling SubClass method
 obj1.display();
 //Creating object 2 using parameterized constructor
 SubClass obj2 = new SubClass(10);
 obj2.display();
 System.out.println("Main Method Ended");
 }
 }

Output

Constructor of Superclass
Constructor of Subclass
Hello
Constructor of Superclass
Constructor with arg
Hello

1). SubClass constructor can make a call to the superclass constructor using super() method.

2). We can also invoke parameterized constructor of parent class by providing arguments while calling super. For e.g. super(10) would invoke the parametrized constructor [having one integer argument] of parent class. Similarly super(“hi”) would invoke constructor having String argument.

3). super() must be the first statement in constructor otherwise we will get the compilation error message: “Constructor call must be the first statement in a constructor”

4). SubClass constructor can call super class constructor either explicitly or implicitly.

5). If the super() method is inserted by the compiler than it is know as implicit constructor.

6). If the super() method is written by the programmer explicitly than it is know as explicit constructor.

super.method_name to invoke parent class method

super.method_name calls the Overriden method of the parent class. The example shown below will make this concept clear:

Sample Code

class SuperClass{
//Overridden method
 void display(){
 System.out.println("Super class method");
 }
 }
class SubClass extends SuperClass{
//Overriding method
 void display(){
 System.out.println("Child class method");
 }
 void printMsg(){
 //This would call Overriding method
 display();
 //This would call Overridden method
 super.display();
 }
 public static void main(String args[]){
Subclass obj= new Subclass();
 obj.printMsg();
}
 }

Output

Child class method
Parent class method
