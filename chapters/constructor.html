Constructor In Java

Constructor are special methods in the class which are used to initialize the instance member of the class. Constructor method has the same name as that of the class and it should not have and return type.

Every Java Class should have a constructor method defined in it. If not the Java Compiler defines a Default Constructor.

A constructor in Java cannot be abstract, static, final or synchronized. These modifier are not allowed for constructor.


The example of constructor is shown below:

Sample Code

class Car{
String name; //Instance variable
 String model; //Instance variable
 Car(){
 name="";
 model="";
 }
 }

Rules for creating Java Constructor

1). Constructor method name must be same as the Class Name.

2). Constructor must not have any return type.

There are two types of Constructor

1). Default Constructor

2). Parameterized Constructor

Each time a new object is created the constructor method will be invoked

Constructor Invocation Example

Car c = new Car(); //Default Constructor invoked
Car c = new Car(name); //Parameterized Constructor invoked

Constructor Overloading

In a Java Class more then one constructor can be defined with different argument list. Such constructor is knows as "Overloaded Constructor".

The overloaded constructor must differ in argument either in type or in argument length.

Constructor overloading is not much different from the method overloading. In case of method overloading we can have multiple methods with same name but different signature, whereas in Constructor overloading also we can have multiple methods with same name and different signature but only difference is that Constructor doesn't have any return type in Java.

Q. Why do we Overload Constructors ?

The simple answer for this question is "We do Constructor Overloading to construct object in different ways".

Constructor overloading is been used to create object with different initialization.

Example of Constructor Overloading

Sample Code 1

class Sample{
int k;
 double d;
 //Constructor Sample(int)
 Sample(int arg1){
System.out.println("Running Sample(int) constructor");
 k = arg1;
 )
 //Constructor Sample(double)
 Sample(double arg2){
 System.out.println("Running Sample(double) constructor");
 d = arg2;
 }
 //Constructor Sample(int,double)
 Sample(int args1,double args2){
 System.out.println("Running Sample(int,double) constructor");
k = args1;
 d = args2;
 }
 }

class Run{
public static void main (String[] args) throws java.lang.Exception{
 System.out.println("Main Method Started");
 Sample ref1 = new Sample(12);
 Sample ref2 = new Sample(12.12);
 Sampleref3 = new Sample(12,12.12);
 System.out.println("Main Method Ended");
 }
 }

Output

Running Sample(int) constructor
Running Sample(double) constructor
Running Sample(int,double) constructor
