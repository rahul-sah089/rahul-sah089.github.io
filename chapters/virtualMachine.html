Virtual Machine Introduction

"The virtual machine is the software simulation of the machine which can perform operation like a physical machine, which is not having physical existence".

One instance of an operating system along with one or more application running in an isolated partition within the computer. It enables the different operating system to run on the same computer at same time.

Virtual are widely used to run the multiple instance of various operating system,each running it own set of applications.The separate VM instance prevent the application from interfering each other. If one program crashes it doesn't affect the programs in the other VMs. This approach differs from dual-boot or multiboot environment, in which user has to choose only one operating system at startup. All the instances runs on the same machine sharing the same computer resources.


VM are like Machines within the Machine

Each virtual machine functions as if it owns an entire computer.The operating systems in each VM partition are called "guest operating systems," and they communicate with the hardware via the virtual machine monitor (VMM) control program. The VMM "virtualizes" the hardware for each VM.

Although a hot topic in the IT world, virtual machine technology dates back to the 1960s. The terms "virtual machine" and "virtualization" are used synonymously; however, virtualization also refers to other techniques.
