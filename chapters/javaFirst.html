Your 1st Java Program

In this section we will discuss about writing our 1st Java Program. For executing any Java Program we need to follow this steps

1). Install the JDK from the oracle website. You will find the setup here

2). Once you had installed the Java on the local system, you need to set the path of jdk/bin directory. Know How to Set the Local Environment Variable

3). Create the Java Program

4). Compile and run the created Java Program

Creating Hello World Program

Let us write the Java Program as shown below

Hello World Example

class Example{
public static void main(String[] args){
 System.out.println("Hello World");
 }
 }

Save this file as Example.java

To Compile:javac Example.java

To Execute:java Example

Output

Hello World

Understanding the 1st program

Let's us understand the meaning of the class,public, static, void, String[] args, System.out.println() method

1).class:This is used to declare the class.

2).public:This is the access modifier of the class. Public means that it is accessable to all.

3).static:This static keyword is been used to indicate that the method can be invoked without creating the object of it. It is known as static references. So since the main method is the 1st method to be loaded by the JVM, the JVM need not create the object to access this method. Thus it saves memory.

4).void:This is the return type of the main method. void means that the main method doesn't returns anything from the itself.

5).String[] args: It is used for passing command line arguments.

6).System.out.println(): It is been used to print statement.

How to write program and execute it?

After you had downloaded and installed the JVM and set it's path in the System variable path, you can follow this steps to write and exectute the programs:

1). Open Notepad: start menu -> All Programs -> Accessories -> notepad

2). Write the sample code in it.

3).When you had written the above code in the notepad, save in as Example.java.

4).To compile the code open the command prompt. To open command prompt Click start then "In search program and files" input field enter "cmd" and hit "Enter"

4).Once the command prompt opens up, change the path to current working directory.

5).Use javac command followed by the name of the Class Name ex: Example.java to compile the Java Code. It creates the Example.class file of it.

6).Use java command followed by the class name without the .java extension.
