This keyword In Java

this is the special keyword which can be used to represent current object or instance of the class.

This keyword contain the current class object address.

It can also be used to differentiate between the instance variable and local variable.


Important point about this keyword in Java

The following are few important point related to "this" keyword in Java:

1). this keyword represent current instance of class.

2). this keyword can be used to call overload constructor in java. If used it must be the 1st statement in constructor. this() can call one argument constuctor with approriate parameter. here is an example of using this() for constructor chaining:

Example of this keyword to call constructor in Java

class Loan{
private double interest;
 private String type;

 public Loan(){
 this(“personal loan”);
 }

 public Loan(String type){
 this.type = type;
 this.interest = 0.0;
 }
 }

3). If member variable and local variable name conflict than this can be used to refer member variable. Here is an example of this with member variable:

Sample Code



public Loan(String type, double interest){
this.type = type;
this.interest = interest;
}

Here local variable interest and member variable interest conflict which is easily resolve by referring member variable as this.interest

4).this is a final variable in Java and you can not assign value to this. this will result in compilation error:

this = new Loan(); //cannot assign value to final variable

5) We can call the method of the same class using this keyword.

Sample Code

public String getName(){
 return this.toString();
 }

6). this can be used to return object. this is a valid return value. here is an example of using as return value.

Sample Code

public Loan getLoan(){
return this;
}

7). this can be used to refer to static member in Java as well but it is discouraged to do so. As per the best practices it should be used to refer to the non-static member of the class.

8). this can also be passed as method parameter since it represent current object of class.
