Inheritance In Java(IS-A)

Inheritance is one of the feature of the Object Oriented Programming that allows a class to use the properties and methods of another class.

In other words the derieved class inherits the states and behaviours from the base class. The derived class is also know as subclass and the base class is also know super-class. The derieved class can add it's own additional variable and methods. These addition variable and methods differentiates the derieved class from the base class.

A superclass can have any number subclass. But a subclass will have only one superclass. This is because Java doesn't support multiple inheritance.

The superclass and subclass have "is-a"relationship between them. Let's look a look in the example below:


Inheritance Example

Consider a class called Shape. Here Shape is the base class from which other sub classes can be derieved. The base class in this case can be rectangle, circle,square etc

Base Class Sample Code

public class Shape{
public double area(){
 return 0;
//Since area() is the generic function for all the other shapes
 //It's implementation consist of only return value 000000
 }
 }

Class Shape is inherited by Circle with is a shape.

The method area is defined in the base class and the implementation is been overriden in the base class method area(). As shown in the example below.

class Circle extends Shape{ //"The extends" keyword is been used to tell the JAVA that the Circle class inherites the functionality parent class Circle
 private static final double PI = Math.PI; //Instance Variable of the Class
 private double diameter; //Instance Variable of the Class
 public double area(){ //Class Method implementation
 double radius = diameter / 2.0;
 return PI * radius * radius;
 }
 }

Another Example of Inheritance

Let's consider a superclass Vehicle. Different vehicles have different features and properties however there are few of them common to all. The common features can be speed,color,size,mileage etc. Hence we can create a class "Vehicles" with state and behaviour as the common to all vehicles. Any types of vehicles can extends from the "Vehicle" class and it can have it's own functionalities too. This subclass can also be extends further and can have it's own base classes. Ex From Vehicles can have sub class as Car and Car can have sub class as Sport's Car.

Sample Code

//A class to display the attribute of the vehicles
 class Vehicles{
String color;
 int speed;
 int size;
 void attributes(){
 System.out.println("Color : " + color);
 System.out.println("Speed : " + speed);
 System.out.println("Size : " + size);
 }
 }
 //A subClass that extends for Vehicles
 class Car extends Vehicles{
 int CC;
 int gears;
 void attributesCar(){
 System.out.println("Color of Car : " + color);
 System.out.println("Speed of Car : " + speed);
 System.out.println("Size of Car: " + size);
 System.out.println("CC of Car : " + speed);
 System.out.println("No of Gears in Car:" + size);
 }
 }

public class Test{
public static void main(String[] args){
 Car ref1 = new Car();
 ref1.color = "Blue";
 ref1.speed = 200;
 ref1.size = 22;
 ref1.CC = 250;
 ref1.gears = 6;
 ref1.attributesCar();
 }
}

The Output Of the Code

Color of Car : Blue
 Speed of Car : 200
 Size of Car : 22
 CC of Car : 250
 No of Gears in Car : 6

Type of Inheritance

The types of inheritance in Java in as follows:

1). Single Inheritance

2). Multiple Inheritance

3). Multilevel Inheritance

4). Hierarchial Inheritance

5). Hybrid Inheritance

1) Single Inheritance

It is the simplest form of the inheritance to understand. When a single class inherits from about base class then it is known as Single Inheritance.

The diagram alongside shows the simple example of Single Level Inheritance.

Here a class B inherits from class A and class C inherits from class B.

Here A is the parent class. B is the child class of A and C is the child class of B.




Sample Code

Class A{
public void methodA(){
 System.out.println("Base class method");
 }
 }

Class B extends A{
public void methodB(){
 System.out.println("Child class method");
 }
 public static void main(String args[]){
 B obj = new B();
 obj.methodA(); //calling super class method
 obj.methodB(); //calling local method
 }
 }

2) Multiple Inheritance

In multiple inhertance concept one class can extends from more than one base class. Though Java doesn't support multiple inheritance, the multiple inheritance can be achieved using interface. Java doesn't support multiple inheritance because it leads to the "Diamond problem" which will be discussed in further sections. In multiple inheritance the derived class has to manage the dependencies on two base class.

Most of the OOPs language like Small talk, Java , C# do not support Multiple inheritance. Multiple inheritance is supported in C++.

The diagram alongside shows class D extending from both class B and class C.



3).MultiLevel Inheritance

In multiLevel iheritance a class inherits from the derived class such that derived class act as the base class of the new class.

In the digram shown alongside class Manager extends from class Employee and Employee class acts as the base class for class Director.



Example of the MultiLevel Inheritance is shown below:

Sample Code

class FirstLevelCategory{
public void addItemToCart1(){
 System.out.println ("addItemToCart1() method been called");
}
 }
class SecondLevelCategory extends FirstLevelCategory{
public void addItemToCart2(){
 System.out.println("addItemToCart2() method been called");
 }
 }

class ThirdLevelCategory extends SecondLevelCategory{
public void addItemToCart3{
 System.out.println("addItemToCart3() method been called");
 }
 }

class Test{
public static void main(String[] args){
 System.out.println("Main Method Started");
 ThirdLevelCategory obj = new ThirdLevelCategory();
 obj.addItemToCart1(); //calling grand parent class method
 obj.addItemToCart2(); //calling parent class method
 obj.addItemToCart3(); //calling local method
 System.out.println("Main Method Ended");
 }
 }


4).Hierarchial Inheritance

In such type of inheritance one class is inherited by many subclasses.

In the diagram alongside Class C and Class D inherits from the same class Class B. This is an example of the hiecarichal inheritance.

The sample code for the Hierarchial Inheritance is shown below:

Sample Code

Class A{
public void methodA() {
 System.out.println("method of Class A");
 }
 }
Class B extends A{
public void methodB(){
 System.out.println("method of Class B");
 }
 }
Class C extends A{
public void methodC(){
 System.out.println("method of Class C");
 }
 }
Class D extends A{
public void methodD() {
 System.out.println("method of Class D");
 }
 }
Class MyClass{
public void methodB(){
 System.out.println("method of Class B");
 }
 public static void main(String args[]){
 B obj1 = new B();
 C obj2 = new C();
 D obj3 = new D();
 obj1.methodA();
 obj2.methodA();
 obj3.methodA();
 }
 }

5). Hybrid Inheritance

Hybrid Inheritance is the combination of the Single and Multiple Inheritance. The diagram alongside shows the example of the Hybrid Inheritance. In the above example class B and class C in the derived class of the A, which itself acts as the parent class for the class D. It is possible to acheive Hybrid Inheritance in Java using Interfaces. Using Interface we can have multiple as well as hybrid interface.
