The If-then and If-then-Else Statement

The If-Then Statement

It is the most basic of all the control flow statement. It tells the program to execute a certain section of the code if it evaluates to true.

If we want to ring an alarm bell when the status is true then we can implement it as shown below.

Computer Code

void ringAlarm(){
if(status == true){
System.out.println("Ring the Alarm");
}
}

The If-Then-Else Statement

The if-Then-Else statement provides a secondary path of execution when "if" clause evaluates to false.

We can apply if-Then-Else in the above example as follows:

Computer Code

void ringAlarm(){
if(status == true){
System.out.println("Ring the Alarm");
}
else{
System.out.println("Don't ring the Alarm");
}
}

The While and Do-While Statement

While statement execute a particular block of the code until the condition is true.

while(expression){
statement(s);
 }

The while statement evaluate expression, which must return a boolean value. Based on the value begin returned by the expression it execute the block of statement in it. If the expression is true, it execute the block of code. If the expression evaluates to false it comes out of the loop.

Printing the Value from 1 to 10 using while loop

class WhileDemo {
public static void main(String[] args){
 int count = 1;
 while (count < 11) {
 System.out.println("Count is: " + count);
 count++;
 }
 }
 }

We can implement infinite loop using the program shown below:

Program to go to infinite loop

while(true){
//statement goes here
}
There is another version of the while loop i.e; do-while loop. Do-while loop evaluate the expression at the bottom of the loop instead of top. Therefore it evaluate the expression atleast once as shown in the DoWhileLoop program below:

The template of do-while loop

class DoWhileLoop{
public static void main(String[] args){
 int count = 1;
 do{
 System.out.println("The Count Value is "+count);
 count++;
 }
 while(count<11);
 }
 }

The For Statement

The for statement provides a compact way to iterate over a particular range of value. It is a way to repeatedly loops until a particular condition is satisfied.

The template of the fo loop is shown below:

Template of for loop

for(initialization; condition;increment/decrement){
statement(s)
 }
1).Intialization: It is used to initialize the loop. It set the initial value of the variable so that the loop can iterate through it
2). Condition: It is the condition until which the loop will iterate.
3). Increment/Decrement : It is used to increment/decrement the value of the value through which the loop iterates through.
The template of the for loop is shown below:

Template "for" Loop

class ForDemo {
for(int i=1; i<11; i++){
 System.out.println("Count is: " + i+";");
 }
}
}

Branching Statement

The break Statement

The break statement are of two types:

1).Labelled

2). UnLabelled

We had seen the UnLabelled form in the previous example of the switch statement.

We can use Unlabelled break to terminate a for,while or do-while loop,as shown in the example below:

The example of the Unlabelled break statement is shown below

This program search for the number 12 in the array. When it is found then the for loop is been terminated using break statement. Control statement then transfer the statement after the for loop

Example:

class BreakDemo {
public static void main(String[] args) {

 int[] arrayOfInts = { 32, 87, 3, 589, 12, 1076, 2000, 8, 622, 127 };
 int searchfor = 12;

 int i;
 boolean foundIt = false;

 for (i = 0; i < arrayOfInts.length; i++) {
 if (arrayOfInts[i] == searchfor) {
 foundIt = true;
 break;
 }
 }

 if (foundIt) {
 System.out.println("Found " + searchfor + " at index " + i);
 } else {
 System.out.println(searchfor + " not in the array");
 }
 }
 }

The output of the above program

Found 12 at 1,0

The Continue Statement

The continue statement skips the current iteration of a for,while,do-while loop. The program shown below shows the example of the continue statement. In this program the character in the given string is been iterated and if the character "p" is not found it skips the rest of the loop and proceed to the next character in the String.

Example of "Continue" Statement

class ContinueDemo {
public static void main(String[] args) {

 String searchMe = "peter piper picked a " + "peck of pickled peppers";
 int max = searchMe.length();
 int numPs = 0;

 for (int i = 0; i < max; i++) {
 // interested only in p's
 if (searchMe.charAt(i) != 'p')
 continue;

 // process p's
 numPs++;
 }
 System.out.println("Found " + numPs + " p's in the string.");
 }
 }

The return Statement

The last of the branching statement is the return statement.

The return statement is been used to return the control from the invoked method to where method was invoked. There are two types of return statement. i.e; the return with some value and the return with void

return ++count;

To return a value from the return simply put the value after it.The data type of the returned value must match the type of the method's declared return value. When a method is declared void, use the form of return that doesn't return a value.

return;

Summary of Control Flow Statements

The if-then statement is the most basic of all the control flow statements. It tells your program to execute a certain section of code only if a particular test evaluates to true. The if-then-else statement provides a secondary path of execution when an "if" clause evaluates to false. Unlike if-then and if-then-else, the switch statement allows for any number of possible execution paths. The while and do-while statements continually execute a block of statements while a particular condition is true. The difference between do-while and while is that do-while evaluates its expression at the bottom of the loop instead of the top. Therefore, the statements within the do block are always executed at least once. The for statement provides a compact way to iterate over a range of values. It has two forms, one of which was designed for looping through collections and arrays.
